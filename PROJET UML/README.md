# Projet UML - Modélisation & Génération de code Python

Projet réalisé dans le cadre du cours UML à l'ISEP Diamniadio.
Chaque sujet part d'un diagramme de classes / cas d'utilisation / séquence (réalisé sur draw.io) et est implémenté en Python orienté objet (encapsulation, composition, classes abstraites, classes d'association).

## Structure du projet

| Sujet | Thème | Fichiers |
|-------|-------|----------|
| Sujet 1 | Gestion des projets étudiants | Diagramme de classes, cas d'utilisation, séquence + `sujet1_projets_etudiants.py` |
| Sujet 2 | Gestion d'une auto-école | Diagrammes de classes, cas d'utilisation, séquences (autorisation examen, séance) + `sujet2_auto_ecole.py` |
| Sujet 3 | Gestion de revues et articles | Diagrammes de classes, cas d'utilisation, séquences (parution, requête auteurs) + `sujet3_revues_articles.py` |
| Sujet 4 | Sécurité sociale | Diagrammes de classes, cas d'utilisation, séquences (consultation, remboursement) + `sujet4_securite_sociale.py` |
| Sujet 5 | Gestion des masters | Diagrammes de classes, cas d'utilisation, séquences (directeur labo, rattacher cours) + `sujet5_gestion_masters.py` |

## Technologies

- **Python 3** (Programmation Orientée Objet)
- **draw.io** pour les diagrammes UML (classes, cas d'utilisation, séquence)
